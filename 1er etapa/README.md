
## Primer etapa
### Pasos para ejecutar el programa `secuencial.cpp`
#### 1. Compilar programa
```bash
g++ secuencial.cpp -o programa
```
#### 2. Ejecutar programa
```bash
./programa M R N
```
M = filas de la matriz A
R = columnas de la matriz A y filas de la matriz B
N = columnas de la matriz B
